Tree Style Tab

作为Firefox最无可替代的扩展，最好用的树状标签页，可以直接css代码更改外观，有大量子扩展增强功能。

附上自己用的式样，主要目的是追求最大效率利用树状栏，紧凑效果。

扩展推荐

Tree Style Tab Mouse Wheel https://addons.mozilla.org/en-US/firefox/addon/tree-style-tab-mouse-wheel/

鼠标悬停后滚轮切换标签，一般国产浏览器才会带的功能，Edge、Chrome都无法做到，最有用的功能。配上罗技anywhere 3的电磁滚轮，非常适合开几十个标签刷论坛查资料。

TST Colored Tabs https://addons.mozilla.org/en-US/firefox/addon/tst-colored-tabs/

彩色化标签，主要可以正则上色，比如把taobao相关域名上橙色，weibo上红色，提醒自己后台开着这些负载重且会在后台不断请求网络的流氓页面。

TST Bookmarks Subpanel https://addons.mozilla.org/zh-CN/firefox/addon/tst-bookmarks-subpanel/

在TST标签栏下面展示书签栏，有效利用空余面积。实际上我书签放在地址栏左边，基本不会用到这个。

/* 新建的未读标签红色表示 */
tab-item.unread .label-content {
  color: #D00000;
}

/* 标签关闭按钮只有在鼠标悬停时显示 */
.tab:not(:hover) .closebox{
	display:none
}

/* 用favicon图标代替箭头，展开和折叠书签树，有效节约宽度 */
tab-item tab-twisty {
  margin-right: -1em;
  opacity: 0 !important;
  position: relative;
  z-index: 10000;
}

/* 左右宽度更紧凑的调整 */
tab-item-substance {
  border: solid 0px #00000012; 
}
tab-item:not(.pinned) tab-item-substance {
  padding-left: 0px !important; /* !important is required when there are enough tabs to cause a scrollbar */
}

/* 标签高度，25px比较紧凑 */
tab-item {
  --tab-size: 25px !important;
  height: var(--tab-size);
}

/* 隐藏侧边栏的Tree Style Tab标题栏，只有切换到TST标签才会隐藏 */
#sidebar-box[sidebarcommand="treestyletab_piro_sakura_ne_jp-sidebar-action"] #sidebar-header {
  display: none;
}

/* 隐藏收藏夹书签文字，只显示favicon，当鼠标悬停时展开该书签文字 */
#personal-bookmarks .bookmark-item > .toolbarbutton-text { display:none !important; }
#personal-bookmarks .bookmark-item:hover > .toolbarbutton-text { display:block !important; }

/* 隐藏顶部标签页，极细化顶部栏，并缩小关闭放大缩小按钮到右上角 */
#titlebar {margin-bottom: -20px !important;}
#tabbrowser-tabs {visibility: collapse !important;}
#nav-bar {margin-right: 0px !important;}
#TabsToolbar {
  height: 1px !important;
  margin-bottom: 15px !important;
  margin-top: -5px !important;
}
